## 📋 总体项目结构

本项目采用 Monorepo 架构，所有应用服务（前端、后端）都放置在 `apps/` 目录下，以便于统一管理和版本控制。

```
aibidcomposer/
├── apps/                              # 应用服务目录
│   ├── frontend/                      # 前端项目 (React)
│   ├── backend-java/                  # Java后端服务 (Spring Boot)
│   └── backend-python/                # Python后端服务 (FastAPI)
├── docker/                            # Docker通用配置
├── k8s/                               # Kubernetes通用配置
├── scripts/                           # 构建和部署脚本
├── docs/                              # 项目文档
├── .github/                           # GitHub Actions/CI配置
├── .gitlab/                           # GitLab CI配置
├── docker-compose.yml                 # 本地开发Docker Compose
├── docker-compose.prod.yml            # 生产环境Docker Compose
├── README.md                          # 项目说明
├── LICENSE                            # 开源协议
└── .gitignore                         # Git忽略文件
```

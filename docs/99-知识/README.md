## 文档分类说明

### 代码规范

包含项目使用的各种编程语言的代码规范文档，确保代码质量和一致性。

### 竞品调研

包含对市场上同类AI标书工具的详细调研和分析，为产品设计提供参考。

主要竞品包括：
- 钛投标
- 易标AI
- 智能投标平台
- 睿标AI

### 技术调研

包含项目中使用的关键技术的调研报告，如：
- MCP（Model Context Protocol）网络搜索工具
- Exa搜索引擎集成

## 文档更新

所有知识库文档应该遵循以下规范：

1. **文档头部**：包含元信息（创建日期、创建者、状态等）

3. **更新记录**：在文档末尾记录修改历史
4. **引用标注**：重要信息和数据来源应明确标注

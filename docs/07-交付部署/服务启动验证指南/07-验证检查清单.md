# AIBidComposer - 服务启动验证指南 - 验证检查清单

### 必须通过的检查项（Critical）

#### 基础设施（5项）
- [ ] Docker正在运行
- [ ] Docker Compose已安装
- [ ] .env文件已配置
- [ ] 所有必需环境变量已设置
- [ ] 密码符合安全要求（最小长度）

#### 数据服务（5项）
- [ ] PostgreSQL健康检查通过
- [ ] PostgreSQL数据库已创建
- [ ] Elasticsearch集群状态正常（green/yellow）
- [ ] Redis PING返回PONG
- [ ] RabbitMQ Management UI可访问

#### 应用服务（5项）
- [ ] Java服务健康检查返回UP
- [ ] Python AI服务健康检查返回healthy
- [ ] 前端服务可访问（开发环境）
- [ ] Celery Worker运行中
- [ ] Celery Beat运行中

#### 服务间通信（3项）
- [ ] Java可连接PostgreSQL
- [ ] Java可连接Redis
- [ ] Python可连接Elasticsearch

### 建议通过的检查项（Optional）

- [ ] MinIO可访问
- [ ] Swagger UI可访问
- [ ] FastAPI Docs可访问
- [ ] 无ERROR级别日志
- [ ] 所有服务内存使用正常
- [ ] 数据库连接池正常
- [ ] Redis缓存读写正常
- [ ] 文件上传功能正常
- [ ] AI API调用正常

**当前竞品定位**:
- 钛投标: "免费AI标书工具"
- 星火投标: "科大讯飞AI投标辅助工具"
- 智标领航: "DeepSeek多模态AI标书工具"
- 易标AI: "开源AI标书工具"
- 睿标AI: "高安全AI标书工具"
- 快标书: "无字数限制AI标书工具"

**我们的定位**:
```
"基于aidev3的企业级AI标书协作平台"

差异化:
├── 企业级: 面向企业，而非个人

```

### 目标客户细分

#### 主要目标客户(Primary Target)

**大型企业投标团队**:
```
特征:
├── 团队规模: 20-100人
├── 年标书量: 100-500份
├── 痛点: 协作效率、知识沉淀、质量保障
├── 预算: 高(年预算50-200万)
└── 决策周期: 长(3-6个月)

需求:
├── 多人实时协作
├── 审批流程管理
├── 知识图谱沉淀
├── 私有化部署
└── 系统集成

定价: 企业版(1299元/月/团队) + 增值服务

竞争策略:
✅ 我们有优势: 企业级功能全覆盖
⚠️ 竞品劣势: 缺少企业级功能
```

**专业服务机构(标书代写公司)**:
```
特征:
├── 团队规模: 10-50人
├── 年标书量: 500-2000份
├── 痛点: 效率、成本、质量一致性
├── 预算: 中高(年预算30-100万)
└── 决策周期: 中(1-3个月)

需求:
├── 极高效率(批量生产)
├── 模板库管理
├── 质量保障(查重、合规)
├── 客户管理
└── 项目管理

定价: 企业版(1299元/月) + 按量计费

竞争策略:
✅ 我们有优势: 效率+质量保障+协作
⚠️ 竞品劣势: 缺少批量处理、客户管理
```

### 客户优先级策略

```
优先级排序:
1. 大型企业投标团队(高价值、长期、可传播)
2. 专业服务机构(高频、大量、可证明ROI)
3. 中型制造企业(标准客户、规模化)
4. 新兴科技企业(成长潜力、早期采用者)
5. 政府及事业单位(长期、稳定、背书价值)
6. 个人投标用户(量大、低价值、可转化)

阶段策略:
├── 第一年: 聚焦1-2(大型企业+专业机构)
├── 第二年: 拓展3-4(中型企业+科技企业)
└── 第三年: 覆盖5-6(政府+个人)
```

# AI标书智能创作平台 - 数据库设计 - 🗄️ 数据表设计

### 设计原则

1. **命名规范**:
   - 表名: 小写+下划线 (snake_case)
   - 字段名: 小写+下划线 (snake_case)
   - 索引名: idx_{table}_{columns}
   - 外键名: fk_{table}_{ref_table}

2. **通用字段**:
   - `id`: UUID主键
   - `created_at`: 创建时间
   - `updated_at`: 更新时间
   - `deleted_at`: 软删除时间 (可选)
   - `created_by`: 创建人ID
   - `updated_by`: 更新人ID

3. **数据类型**:
   - ID: UUID
   - 时间: TIMESTAMP WITH TIME ZONE
   - 金额: DECIMAL(15, 2)
   - JSON: JSONB
   - 文本: TEXT

### 数据库初始化

```sql
-- ============================================================================
-- AI标书智能创作平台 - 数据库初始化脚本
-- 版本: 1.0
-- 创建时间: 2025-11-14
-- ============================================================================

-- 创建数据库
CREATE DATABASE aibidcomposer
    WITH
    ENCODING = 'UTF8'
    LC_COLLATE = 'zh_CN.UTF-8'
    LC_CTYPE = 'zh_CN.UTF-8'
    TEMPLATE = template0;

\c aibidcomposer;

-- 启用扩展
CREATE EXTENSION IF NOT EXISTS "uuid-ossp";      -- UUID生成
CREATE EXTENSION IF NOT EXISTS "pg_trgm";        -- 模糊搜索
CREATE EXTENSION IF NOT EXISTS "btree_gin";      -- GIN索引
CREATE EXTENSION IF NOT EXISTS "timescaledb";    -- 时序数据

-- 设置时区
SET TIME ZONE 'Asia/Shanghai';
```

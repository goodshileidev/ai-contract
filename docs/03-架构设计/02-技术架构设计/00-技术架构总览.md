# AI标书智能创作平台 - 技术架构总览

> **架构模式**: 混合后端微服务架构
> **后端**: Java Spring Boot (业务) + Python FastAPI (AI)
> **前端**: React 18 + Ant Design Pro

## 🏗️ 整体架构设计

### 架构原则
- **微服务架构**: 模块化设计,便于扩展和维护
- **混合后端**: Java处理业务数据,Python处理AI能力
- **云原生**: 支持容器化部署和弹性扩缩容
- **API优先**: 所有功能通过API提供服务
- **数据驱动**: 基于数据分析优化产品功能

2. 为每个页面设计完整的组件集
3. 自动分配组件ID（避免冲突）
4. 生成符合schema的JSON文件
5. 保持与页面的关联关系

**关键代码特性**:
```python
# ID连续性保证
existing_sections = list(self.section_dir.glob("*.json"))
max_id = max(section['page_section_id'] for section in existing_sections)
self.section_id_counter = max_id + 1

# 自动关联页面
section = {
    "page_section_id": self.section_id_counter,
    "page_id": page["page_id"],  # 自动关联
    "module_id": page["module_id"],  # 继承模块
    # ...
}
```

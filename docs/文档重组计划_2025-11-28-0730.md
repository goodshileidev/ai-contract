## 📝 OpenAPI文件生成计划

### 主OpenAPI文件（openapi.yaml）

```yaml
openapi: 3.0.3
info:
  title: AIBidComposer API
  version: 1.0.0
  description: AI标书智能创作平台API文档
servers:
  - url: https://api.aibidcomposer.com/api/v1
    description: 生产环境
  - url: http://localhost:8080/api/v1
    description: 开发环境（Java服务）
  - url: http://localhost:8001/api/v1
    description: 开发环境（Python AI服务）

# 引用各个模块的API定义
paths:
  # 认证授权
  /auth:
    $ref: './auth.yaml#/paths/~1auth'
  # 项目管理
  /projects:
    $ref: './projects.yaml#/paths/~1projects'
  # ... 其他模块
```

# AI标书智能创作平台 - 开发环境配置指南 - 🛠️ 开发环境概览

### 环境要求
```yaml
系统要求:
  操作系统:
    - 开发环境: macOS 12+ / Windows 11 / Ubuntu 20.04+
    - 生产环境: Linux (Ubuntu 20.04+ / CentOS 8+)
    - 测试环境: 支持以上所有操作系统

  硬件要求:
    CPU: 4核心以上（推荐8核心）
    内存: 16GB以上（推荐32GB）
    硬盘: 500GB SSD（推荐1TB NVMe）
    网络: 稳定网络连接（100Mbps以上）

  开发工具:
    - IDE: VS Code / WebStorm / PyCharm
    - 版本控制: Git 2.30+
    - 容器化: Docker Desktop / Docker Engine
    - API测试: Postman / Insomnia
    - 数据库工具: pgAdmin / DataGrip
```

### 开发团队协作环境
```yaml
代码仓库:
  主要仓库: GitHub / GitLab
    - 前端代码: React应用
    - 后端代码: FastAPI应用
    - 配置文件: Docker、CI/CD配置
    - 文档: 项目文档、API文档

  分支策略:
    - main: 生产环境代码
    - develop: 开发环境代码
    - feature/*: 功能开发分支
    - hotfix/*: 紧急修复分支

  协作流程:
    - 功能开发: develop → feature/* → develop → main
    - 紧急修复: main → hotfix/* → main
    - 代码审查: 所有分支必须经过代码审查
    - 自动化测试: CI/CD自动运行测试
```

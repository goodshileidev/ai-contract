# AI标书智能创作平台 - 数据库设计 - 📋 数据库概览

### 数据库选型

```yaml
主数据库:
  类型: PostgreSQL 14+
  目的: 关系型数据存储
  特性: ACID、JSONB、全文搜索、时序数据
  访问: Java Spring Boot服务主要负责CRUD操作

缓存数据库:
  类型: Redis 7+
  目的: 缓存、会话存储、消息队列
  访问: Java和Python服务共享

搜索与向量检索引擎:
  类型: Elasticsearch 8+
  目的:
    - 全文搜索
    - 向量嵌入存储和相似度搜索（替代Pinecone）
    - 关系建模（替代Neo4j基础功能）
    - 日志分析
  访问: 主要由Python AI服务访问

对象存储:
  类型: MinIO
  目的: 文件存储（文档、图片等）
  访问: Java和Python服务共享
```

### 数据库架构

```
PostgreSQL数据库:
├── public schema (公共模式)
│   ├── 用户与权限域 (9张表)
│   ├── 项目域 (6张表)
│   ├── 标书域 (8张表)
│   ├── 模板域 (5张表)
│   ├── 企业能力域 (7张表)
│   ├── AI服务域 (5张表)
│   ├── 协作域 (4张表)
│   ├── 审批域 (4张表)
│   └── 审计与日志域 (3张表)
│
└── extensions
    ├── uuid-ossp (UUID生成)
    ├── pg_trgm (模糊搜索)
    └── timescaledb (时序数据)
```

# AIBidComposer - 服务启动验证指南 - 验证范围

本验证涵盖以下服务：

### 基础数据服务（5个）
1. **PostgreSQL 15** - 关系型数据库
2. **Elasticsearch 8.11** - 搜索引擎和向量数据库
3. **Redis 7** - 缓存和会话存储
4. **RabbitMQ 3** - 消息队列
5. **MinIO** - 对象存储

### 应用服务（5个）
6. **Backend Java** - Spring Boot业务API（端口8080）
7. **Backend Python** - FastAPI AI服务（端口8001）
8. **Frontend** - React开发服务器（端口5173）
9. **AI Worker** - Celery异步任务处理
10. **Celery Beat** - 定时任务调度

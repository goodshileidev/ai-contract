## 概述

本文档是 **AIBidComposer（AI标书智能创作平台）** 项目开发任务计划的**总览索引**，用于：
- 提供项目任务的全局视图
- 跟踪各模块的完成进度
- 导航到各技术栈的详细任务计划

**详细任务计划已按技术栈拆分为以下子文件**：
- [基础设施任务计划](./基础设施实现计划.md)
- [Java Spring Boot 服务任务计划](./Java服务实现计划.md)
- [Python FastAPI AI 服务任务计划](./Python-AI服务实现计划.md)
- [React 前端任务计划](./前端实现计划.md)

### 项目架构特点

本项目采用**混合后端微服务架构**：
- **Java Spring Boot 服务**（端口 8080）- 数据维护、业务逻辑、CRUD
- **Python FastAPI 服务**（端口 8001）- AI能力、大模型调用
- **React + Ant Design Pro 前端** - 企业级用户界面

### 任务组织原则

1. **按技术栈分类**：Java模块、Python AI模块、前端模块、基础设施
2. **按功能模块分类**：对应8大核心功能模块
3. **按优先级分类**：P0-P3，确保核心功能优先交付

## 风险和问题

### 当前风险

#### 技术风险
1. **AI模型稳定性**
   - 描述: GPT-4 API 可能出现限流或服务不稳定
   - 影响: 高
   - 缓解措施: 实现重试机制，准备 Claude 作为备用模型

2. **向量检索性能**
   - 描述: Elasticsearch 向量检索性能可能不够
   - 影响: 中
   - 缓解措施: 优化索引配置，必要时使用专用向量数据库

#### 进度风险
1. **团队规模**
   - 描述: MVP 开发任务量较大，3个月时间紧张
   - 影响: 高
   - 缓解措施: 优先完成核心流程，非核心功能延后到 v2.0

## 使用说明

### 任务状态标记
- ⏸️ **待开始**: 任务已计划但未开始
- 🔄 **进行中**: 任务正在开发
- ✅ **已完成**: 任务已完成并通过验证
- ⚠️ **已阻塞**: 任务被阻塞，无法继续

### 更新频率
- **每日**: 更新进行中任务的进度
- **每周**: 回顾本周完成情况，规划下周任务
- **每月**: 回顾迭代进展，调整计划和优先级

### 工作流程

#### 1. 选择任务模块
根据你的职责选择对应的任务计划文档：
- DevOps/全栈开发 → [基础设施实现计划.md](./基础设施实现计划.md)
- Java 后端开发 → [Java服务实现计划.md](./Java服务实现计划.md)
- Python AI 开发 → [Python-AI服务实现计划.md](./Python-AI服务实现计划.md)
- 前端开发 → [前端实现计划.md](./前端实现计划.md)

#### 2. 在模块文档中选择具体任务
查看详细的子任务清单和技术实现要点

#### 3. 使用 TodoWrite 创建会话任务
将选中的任务拆分为更细粒度的开发步骤

#### 4. 执行开发
按照 TodoWrite 任务清单逐步完成

#### 5. 更新进度
- 更新模块文档中的任务完成状态
- 本总览文档的进度会根据各模块文档自动汇总

#### 6. 提交代码
使用规范的 Git 提交格式，包含需求编号

### 常用命令速查

#### 开发环境启动
```bash
# 启动所有服务
docker-compose up -d

# 启动 Java 服务
cd apps/backend-java
mvn spring-boot:run

# 启动 Python AI 服务
cd apps/backend-python
uvicorn app.main:app --reload --port 8001

# 启动前端
cd apps/frontend
npm run dev
```

#### 数据库迁移
```bash
# Java 服务（Flyway 自动执行）
mvn flyway:migrate

# 手动执行特定版本
mvn flyway:migrate -Dflyway.target=2
```

# Java 单元测试
mvn test

# Python 测试
pytest

npm test

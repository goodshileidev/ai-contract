## 📖 核心内容摘要

### CLAUDE.md 核心要点

**技术栈原则**:
- ✅ 后端：纯 Java 技术栈（Spring Boot 3.2 + Spring AI）
- ✅ 前端：React 18 + TypeScript + Ant Design Pro
- ✅ 数据库：PostgreSQL + Elasticsearch + Redis
- ❌ 禁止：引入 Python、Node.js 等其他后端语言

**项目铁律（10条）**:
1. 简体中文优先
2. 纯 Java 后端技术栈
3. 文档集中管理
4. 文档目录标准化
5. 文档元信息完整
6. 修改历史可追溯
7. 源码需求标注
8. 完整功能必须提交 Git
9. 数据必须可追溯
10. 诚实透明优先

**开发工作流**:
```
确认需求 → 架构设计 → 详细设计 → 编码实现 → 单元测试
→ 代码审查 → 集成测试 → 文档更新 → Git提交
```

# AIBidComposer - 服务启动验证指南 - 快速验证

### 方法1: 自动化脚本验证（推荐）

```bash
# 给脚本添加执行权限
chmod +x scripts/verify-services.sh

# 验证开发环境
./scripts/verify-services.sh

# 验证生产环境
./scripts/verify-services.sh --prod
```

**脚本功能**:
- ✅ 自动检查所有15项服务状态
- ✅ 测试服务间通信
- ✅ 显示详细的健康检查结果
- ✅ 提供通过率统计
- ✅ 显示所有服务访问地址

**预期输出示例**:
```
========================================
验证摘要
========================================
总检查项: 35
通过: 35
失败: 0
通过率: 100%

✓ 所有检查均通过！系统运行正常。
```

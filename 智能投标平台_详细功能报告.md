## 🎯 核心功能

### 1. 智能标书生成 🤖
- **AI驱动生成**: 基于LangChain + LangGraph平台的智能标书生成
- **模板化处理**: 支持多种标书模板和格式
- **内容智能化**: AI自动分析招标要求并生成相应内容
- **质量优化**: 智能优化标书内容质量和专业度

### 2. 多格式文档预览 📄
- **Word文档**: 支持Word格式的标书文档预览
- **Excel表格**: 支持Excel格式的数据表格预览
- **PDF文档**: 支持PDF格式的文档预览
- **实时预览**: 在线实时预览生成的标书内容

### 3. 用户认证系统 🔐
- **多种登录方式**: 支持多种用户登录和认证方式
- **账户管理**: 完整的用户账户注册、登录、管理功能
- **安全验证**: 多层次的安全验证机制
- **权限控制**: 基于角色的用户权限管理

### 4. 验证服务服务 ✉️
- **短信验证**: 手机短信验证码服务
- **邮件验证**: 电子邮件验证服务
- **身份确认**: 用户身份多重确认机制
- **安全保护**: 账户安全和反欺诈保护

## 🛠️ 技术架构

### 前端架构
```
Vue 3 + TypeScript + Vite
├── UI组件库: Element Plus
├── 状态管理: Vuex/Pinia

├── HTTP客户端: Axios
├── 构建工具: Vite
└── 类型检查: TypeScript
```

```
Django 5 + Ninja API Framework
├── 数据库: PostgreSQL
├── 缓存: Redis
├── API框架: Django Ninja
├── 认证系统: Django Authentication
├── 文件处理: Django File Storage
└── 异步任务: Celery (可能)
```

### AI技术栈
```
LangChain + LangGraph 深度集成
├── 大语言模型集成:
│   ├── OpenAI GPT-3.5/4.0
│   ├── Claude系列模型
│   ├── 国产大模型 (通义千问、文心一言等)
│   └── 自定义模型接口
├── 工作流编排: LangGraph
│   ├── 状态图设计: 复杂投标流程建模
│   ├── 条件分支: 智能决策节点
│   ├── 并行处理: 多任务并发执行
│   ├── 错误处理: 异常恢复机制
│   └── 状态持久化: 工作流状态保存
├── 提示词管理:
│   ├── 模板引擎: 动态提示词生成
│   ├── 上下文管理: 长文本上下文维护
│   ├── 参数化: 可配置的提示词参数
│   └── 版本控制: 提示词版本管理
├── 文档处理: LangChain Document Loaders
│   ├── PDF解析: 高精度PDF文本提取
│   ├── Word处理: Word文档内容解析
│   ├── Excel处理: 表格数据结构化
│   ├── 网页抓取: 在线内容获取
│   └── OCR集成: 图片文字识别
├── 向量存储:
│   ├── ChromaDB: 本地向量数据库

│   ├── FAISS: 高性能向量检索
│   └── 向量嵌入: 文本向量化处理
└── 内容生成: LangChain Chains
    ├── 顺序链: 多步骤处理流程
    ├── 路由链: 条件性内容生成
    ├── 记忆链: 上下文记忆功能
    ├── 转换链: 数据格式转换
    └── 总结链: 内容摘要生成
```

## 🚀 部署方案

### Docker Compose 部署
```yaml
# docker-compose.yml (推测结构)
version: '3.8'
services:
  web:
    build: .
    ports:
      - "8000:8000"
    environment:
      - DEBUG=False
      - DATABASE_URL=postgresql://user:pass@db:5432/dbname
      - REDIS_URL=redis://redis:6379
    depends_on:
      - db
      - redis

  db:
    image: postgres:15
    environment:
      - POSTGRES_DB=intelligent_bid
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=password
    volumes:
      - postgres_data:/var/lib/postgresql/data

  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"

  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx.conf:/etc/nginx/nginx.conf
      - ./static:/var/www/static
    depends_on:
      - web

volumes:
  postgres_data:
```

## 🔧 开发和部署指南

### 开发环境搭建
```bash
# 1. 克隆项目
git clone https://github.com/lanhui100/intelligent-bid.git
cd intelligent-bid

# 2. 后端环境
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# 3. 前端环境
cd frontend
npm install
npm run dev

# 4. 数据库迁移
cd ../backend
python manage.py migrate
python manage.py createsuperuser
```

### 生产部署
```bash
# 1. Docker部署
docker-compose up -d

# 2. 数据库初始化
docker-compose exec web python manage.py migrate

# 3. 收集静态文件
docker-compose exec web python manage.py collectstatic

# 4. 创建超级用户
docker-compose exec web python manage.py createsuperuser
```
